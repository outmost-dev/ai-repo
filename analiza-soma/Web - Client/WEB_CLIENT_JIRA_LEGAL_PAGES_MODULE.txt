================================================================================
STORY: LEGAL PAGES MODULE
================================================================================

Title: Legal Pages Module

Description:
Modulul Legal Pages gestionează afișarea paginilor legale statice ale platformei,
inclusiv politica de confidențialitate, politica de cookie-uri și termene și condiții.

Toate paginile legale sunt accesibile public (fără autentificare) și folosesc
o structură comună cu meniu lateral pentru navigare între secțiuni.

Locație cod sursă:
- Pages: web/src/pages/legal/
- Components: web/src/components/pages/legal/

Componente implementate: 4
- Politică de Confidențialitate (politica-de-confidentialitate.tsx)
- Politică de Cookie (politica-de-cookie.tsx)
- Termene și Condiții (termene-si-conditii.tsx)
- AsideMenu (navigation component)

Pattern comun:
- Layout public (UserRoles.GUEST)
- Grid responsiv: AsideMenu (8 cols) + Content (16 cols)
- FloatButton.BackTop pentru scroll to top
- Typography pentru conținut legal extins
- PageHelmet pentru SEO metadata

Tech Stack Current:
- Next.js 15.2.4 pages router
- React 18.2.0
- Ant Design 5.24.2 (Layout, Row, Col, Typography, FloatButton)
- next/router pentru active route detection

Tech Stack Target:
- Nuxt 3 (pages router)
- Vue 3.4+ Composition API
- Ant Design Vue 4.x (Layout, Row, Col, Typography, FloatButton)
- Vue Router 4.x pentru active route detection

Dependencies:
- AppLayout (shared component)
- PageHelmet (shared component)
- PageTitle (shared component)
- UserRoles enum

Notes:
1. Toate paginile sunt 100% statice - fără API calls sau state management
2. Conținutul legal este hard-coded în componente (nu vine din CMS)
3. GDPR compliance: politica de confidențialitate include informații despre
   colectare date, drepturi utilizatori, și integrări third-party
4. AsideMenu folosește router.asPath pentru active state highlighting
5. Responsive: pe mobile (xs) meniul apare deasupra conținutului (24 cols fiecare)

================================================================================
TASK 1: POLITICĂ DE CONFIDENȚIALITATE PAGE
================================================================================

Title: Politică de Confidențialitate Page

Story: Legal Pages Module

Description:

Business Logic:
Pagina Politică de Confidențialitate afișează informațiile legale despre
colectarea, procesarea și protecția datelor personale ale utilizatorilor,
conform GDPR. Include:

1. Informații generale despre protecția datelor (GDPR compliance)
2. Tipuri de date colectate (nume, email, adresă, telefon, etc.)
3. Scopuri procesării datelor (executare servicii, marketing, analiză)
4. Partajare date cu terți (Facebook, LinkedIn, Google AdWords)
5. Drepturi utilizatori (acces, rectificare, ștergere, portabilitate)
6. Perioade de păstrare date
7. Securitate și cookies
8. Contact pentru Data Protection Officer

Pagina este accesibilă public (fără autentificare) și include:
- Meniu lateral cu link-uri către alte pagini legale (AsideMenu)
- Conținut extins cu secțiuni structurate (Typography)
- Scroll to top button (FloatButton.BackTop)
- Meta tags pentru SEO (PageHelmet)

```tsx
// web/src/components/pages/legal/AsideMenu.tsx
import React from 'react';
import Link from 'next/link';
import { useRouter } from 'next/router';

const AsideMenuContent = [
  { title: 'Politică de cookie-uri', slug: 'politica-de-cookie' },
  { title: 'Politică de confidențialitate', slug: 'politica-de-confidentialitate' },
  { title: 'Termene și condiții', slug: 'termene-si-conditii' },
];

export const AsideMenu: React.FC = () => {
  const router = useRouter();

  return (
    <aside>
      <ul className="legal-menu">
        {AsideMenuContent.map((page) => (
          <li key={page.slug} className={router.asPath === `/legal/${page.slug}` ? 'active' : ''}>
            <Link href={`/legal/${page.slug}`}>
              {page.title}
            </Link>
          </li>
        ))}
      </ul>
    </aside>
  );
};
```

Technical Details:

**Data Structure:**
- AsideMenuContent: Array of objects with `title` and `slug`
- Static data defined in component (3 items)

**Active State:**
- Uses `router.asPath` from next/router
- String comparison: `router.asPath === `/legal/${page.slug}``
- Adds `active` CSS class to matching <li>

**Rendering:**
- Maps over AsideMenuContent array
- Each item rendered as <li> with Link component
- Link href constructed: `/legal/${page.slug}`

**Styling:**
- CSS class: "legal-menu" on <ul>
- Active class: "active" on <li>
- Styles defined globally (not in component)
